كيف أحصل على مفتاح YouTube API؟

تحتاج إلى مفتاح API لتشغيل هذا البرنامج. اتبع هذه الخطوات:

  1. اذهب إلى Google Cloud Console
  2. سجل الدخول بحساب Google
  3. أنشئ مشروعًا جديدًا
  4. اذهب إلى "APIs & Services" → "Library"
  5. ابحث عن "YouTube Data API v3" وقم بتفعيله
  6. اذهب إلى "Credentials" → "Create Credentials" → "API Key"
  7. انسخ المفتاح الذي تم إنشاؤه

نصيحة: قم بتقييد المفتاح على "YouTube Data API v3" فقط للحماية.
